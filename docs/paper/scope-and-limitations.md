# Scope and Limitations

## Scope
HariKnows covers the registrar support needs of the university through the following areas:

- Chatbot assistance for common registrar questions.
- Signed-in student access to their own dashboard and request history.
- Registrar document request tracking with the Pending -> Prepared -> Claimed -> Disposed lifecycle.
- FAQ and context management for registrar policies and guidance.
- Student credential management for existing student numbers.
- CSV uploads for registrar bulk updates, including ICTO account files.
- Office routing for relevant registrar-related concerns.

## Limitations
The system does not do the following:

- It does not replace official on-site registrar validation.
- It does not accept online filing for official document requests.
- It does not expose private grades or another student's records.
- It does not provide password recovery or forgotten-password flows.
- It does not automatically notify users by email or push message.
- It does not extend beyond the registrar-related scope defined in the project context.
- It does not make policy decisions that belong to the registrar or related offices.
